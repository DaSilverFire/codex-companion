Install Codex Companion

1. Drag Codex Companion.app onto the Applications shortcut.
2. Replace the older copy if Finder asks.
3. Open Codex Companion from Applications or Spotlight.

The app is already built for Apple silicon and Intel Macs. Xcode, Swift, and a
source checkout are not required. Installing an update preserves your settings,
selected pet, pet packages, task history, and Keychain data.

This release does not modify or re-sign ChatGPT and does not start or replace the
shared Codex app-server.
